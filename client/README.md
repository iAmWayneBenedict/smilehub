# SmileHub

SmileHub

## Installation

Use the package manager [pnpm](https://pnpm.io/installation) to install smilehub.

```bash
npm install -g pnpm
```

or

```bash
npm install -g @pnpm/exe
```

## Run Locally

Install all dependencies

```bash
pnpm install
```

Run the app

```bash
pnpm dev
```
